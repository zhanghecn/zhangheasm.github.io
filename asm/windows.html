# 了解windows API

win32API官网 (opens new window)

  • SetWindowsHookEx HOOK 鼠标 键盘等事件，可监听鼠标键盘等信息

  • GetMessage 获取注册监听的事件

  • GetCursorInfo 获取当前游标位置

  • WindowFromPoint 通过游标点找窗口

  • GetWindowThreadProcessId 获取窗口的进程id

  • OpenProcess 打开进程访问

  • CloseHandle 关闭访问句柄

  • VirtualQueryEx 查询虚拟内存页

  • ReadProcessMemory 读取内存

  • WriteProcessMemory 写内存

  • VirtualAllocEx 提交内存

  • VirtualFreeEx 释放内存
